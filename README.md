# 📝 Python To-Do App

A simple **Command Line To-Do List Application** built using Python.
This program allows users to manage their daily tasks by adding, completing, marking, deleting, and viewing tasks.

---

## 🚀 Features

* Add multiple tasks
* Mark tasks as completed
* View pending tasks
* Mark important tasks
* Delete tasks
* View all tasks (Pending, Completed, Marked)
* Simple command-line menu interface

---

## 📌 Menu Options

```
1. Add Task
2. Add Completed Task
3. View Pending Task
4. Mark Task
5. Delete Task
6. View All Task
7. Exit
```

---

## ⚙️ How It Works

* **Pending Tasks** are stored in a list.
* **Completed Tasks** are moved from pending list to completed list.
* **Marked Tasks** help highlight important tasks.
* Users can delete tasks from pending or completed lists.
* The program runs in a loop until the user chooses **Exit**.

---

## 🧠 Algorithm

1. Start the program
2. Initialize lists for pending, completed, and marked tasks
3. Display menu options
4. Take user input for operation
5. Perform the selected task operation
6. Repeat the menu until the user selects Exit
7. End program
